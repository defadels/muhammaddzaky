Getting Started
This project require NodeJS v12.

Install
Clone this project

> git clone https://github.com/YogaSakti/imageToSticker.git
> cd imageToSticker
Install the dependencies:

> npm install
Usage
run the Whatsapp bot
> npm start
after running it you need to scan the qr
